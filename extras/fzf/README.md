# Bamboo Colors for FZF

## Bamboo Vulgaris

Append colors to `FZF_DEFAULT_OPTS`:

```sh
export FZF_DEFAULT_OPTS="$FZF_DEFAULT_OPTS \
  --color=bg+:#0B342D,bg:#031A16,spinner:#963E4C,hl:#963E4C \
  --color=fg:#81B5AC,header:#963E4C,info:#3E9688,pointer:#4C963E \
  --color=marker:#4C963E,fg+:#4C963E,prompt:#4C963E,hl+:#963E4C \
  --color=selected-bg:#184E45 \
  "
```
